# Illinois Adoption Criteria
**State Board of Education Requirements**
**Source:** Illinois State Board of Education (ISBE)
**Applies To:** All Illinois instructional materials (all subjects)

---

## Overview

**Illinois uses local control for curriculum adoption** with state oversight through ISBE. There is no formal statewide textbook adoption process. All materials must align to Illinois Learning Standards (ILS) and support English Learners.

**Core Principle:** Materials must align to state learning standards (CCSS-based), support English Learners, provide accessibility, and demonstrate quality instructional design.

---

## Key Facts

| Aspect | Illinois |
|--------|----------|
| **Content Standards** | Illinois Learning Standards (ILS) - CCSS-based |
| **Language Standards** | English Learners (WIDA consortium) |

| **Science Standards** | Illinois Learning Standards for Science (NGSS-based) |

## Illinois Learning Standards (ILS)

### Content Standards:

**Mathematics:**
- Illinois Learning Standards for Mathematics
- Based on CCSS-M with Illinois clarifications
- Nearly identical to CCSS-M (95%+ alignment)

**ELA:**
- Illinois Learning Standards for English Language Arts
- Based on CCSS-ELA
- Emphasis on college and career readiness

**Science:**
- Illinois Learning Standards for Science (2016)
- Based on NGSS framework
- Three-dimensional learning (DCIs, SEPs, CCCs)

**Social Science:**
- Illinois Learning Standards for Social Science
- Incorporates C3 Framework

## Local Adoption Process

**Illinois districts independently evaluate and adopt materials:**

**Typical District Criteria:**
- Alignment to Illinois Learning Standards
- Support for English Learners (WIDA)
- Assessment quality
- Digital-print integration
- Cost and professional development
- Accessibility (WCAG 2.1 AA)

**State Assessments Influence Decisions:**
- IAR (Illinois Assessment of Readiness) - Grades 3-8 Math/ELA
- ISA (Illinois Science Assessment) - Grades 5, 8, 11
- SAT - Grade 11 (college readiness indicator)

## Implementation Notes

### When Creating Illinois Math Lessons:

**Content Standards:**
- Tag with Illinois Learning Standards codes
- Example: IL.Math.5.NF.A.1 (equivalent to CCSS.MATH.CONTENT.5.NF.A.1)
- **Practical:** Use CCSS-M tags with note "Adopted in IL as IL.Math.[code]"

**Language Support:**
- Reference WIDA ELD Standards
- Use MLRs (same as all states - math common)
- Provide scaffolds for WIDA proficiency levels (1.0-6.0, commonly grouped as 1-2, 3-4, 5-6)

**Assessment:**
- Same item types as other states (universal)
- IAR alignment for grades 3-8
