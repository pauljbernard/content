# Curriculum QTI Export
Export assessment items and tests to IMS QTI 2.1 format for interoperability across learning management systems.

## CLI: /curriculum.qti-export --assessment 'quiz-001.json' --version '2.1' --output 'qti/'
## Exit Codes: 0=success, 1=invalid assessment, 2=export failed
