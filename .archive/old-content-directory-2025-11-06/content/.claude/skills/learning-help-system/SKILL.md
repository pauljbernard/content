# Learning Help System
Create integrated help and support systems for learning platforms.
## When to Use: LMS help, platform onboarding, feature documentation, user guides
## Elements: Tooltips, walkthroughs, video tutorials, searchable docs, chatbot content
## CLI: /learning.help-system --platform "custom LMS" --features "features.json" --onboarding
## Output: Help documentation, tooltip content, tutorial scripts, search index
## Exit Codes: 0=success, 1=invalid platform, 2=missing feature specs
