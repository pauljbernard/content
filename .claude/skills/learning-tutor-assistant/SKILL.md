# Learning Tutor Assistant
Deliver personalized, adaptive tutoring that meets learners where they are.
## When to Use: Individual support, homework help, concept clarification, skill practice
## Techniques: Multiple explanations, worked examples, scaffolded hints, error analysis
## CLI: /learning.tutor-assistant --concept "quadratic formula" --learner-level "struggling" --explain
## Output: Adaptive explanations, practice problems, hints, remediation
## Exit Codes: 0=success, 1=concept not found, 2=invalid level
