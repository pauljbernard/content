# Learning Microlesson Designer
Design ultra-focused lessons for mobile-first, just-in-time learning.
## When to Use: Mobile learning, performance support, quick tips, refreshers
## Format: Single objective, 3-5 minutes, mobile-optimized, standalone
## CLI: /learning.microlesson-designer --objective "LO-1.1" --duration "3min" --mobile-first
## Output: Micro-content, visuals, knowledge check, mobile layout
## Exit Codes: 0=success, 1=too complex for micro-format, 2=invalid duration
