# Learning Peer Review Designer
Create effective peer assessment that develops evaluation skills and provides feedback.
## When to Use: Writing courses, project-based learning, collaborative work, formative feedback
## Components: Review rubrics, calibration exercises, anonymization, quality checks
## CLI: /learning.peer-review-designer --assignment "essay" --reviewers-per-submission "3" --blind
## Output: Peer review rubric, training materials, review workflow, quality criteria
## Exit Codes: 0=success, 1=invalid assignment type, 2=unrealistic reviewer ratio
