**Vendor Checklists by Content Block v1 (Checklist Only)**

**1. Learning Goals**

**SE**

-   Includes **I Can statement** (student-facing).

-   Uses grade-appropriate, student-friendly language.

-   Bold key math vocabulary.

-   Provides rating options: *I don't understand / I need more practice
    / I've got it.*

**TE**

-   Begins with **START** boilerplate for the Learning Goal routine.

-   Includes at least 3 steps: read aloud, clarify key terms, students
    self-assess.

-   Clarifies new vocabulary using glossary definitions.

-   Ends with instruction to revisit Learning Goal at session end.

**2. Before You Teach**

**TE only**

-   Contains "What should I understand about math?" section (≤420
    characters, 3 bullets).

-   Includes **Prior Learning** (3 bullets, past tense verbs, lowercase
    starts).

-   Focus Standard(s) stated verbatim.

-   ELPS code + text included.

-   Learning Objective + Language Objective provided.

-   Future Connections (3 bullets, "will..." phrasing).

-   Manipulatives list: per student, per group, digital.

**3. Spark Your Learning (SYL)**

**SE**

-   One problem aligned to prerequisite knowledge (≤120 words).

-   Ends with *Show your thinking.*

-   Bold math vocabulary.

-   Includes Spec Type, Description, and Alt Text if art/photo/tech art
    is used.

-   Turn & Talk included with annotation.

**TE**

-   Begins with **START** boilerplate.

-   Includes **MLR6: Three Reads** routine (context, quantities,
    question).

-   Provides Spark Discussions:

    -   Common Error (student example + TG script + 3 ASK Qs).

    -   Deepening Thinking (student example + TG script + 3 ASK Qs).

-   Turn & Talk TG guide included.

-   Includes ML scaffolds at 3 proficiency levels (Emerging, Expanding,
    Bridging).

-   Includes 1 UDL strategy.

-   Vocabulary clarification included.

**4. Review Spark Your Learning**

**SE**

-   Restates Spark Your Learning problem in shortened form.

-   Adds reflection Q (e.g., *How would you check your work?*).

-   Includes Spec Type, Description, Alt Text if visual used.

**TE**

-   Begins with **NOW, review Spark Your Learning**.

-   Includes at least 3 ASK Qs about checking work/operations.

-   Closes with feedback guidance.

-   Multilingual Learner strategy integrated (peer/group sharing,
    respectful conversation).

**5. Quick Check**

**SE**

-   Interaction problems labeled sequentially (Interaction 1, 2...).

-   Each Interaction includes SE Manuscript + annotation.

-   Spec Type + Description included if visual.

-   Bold vocabulary.

**TE**

-   Begins with **NOW, use the Quick Check** boilerplate.

-   Language Objective hook included.

-   Teaching Strategy: Multilingual Learners present with ELPS
    alignment.

-   Optional: UDL or Common Error included if provided in Data Inputs.

**6. Tasks**

**SE**

-   Includes main problem stem.

-   Sub-items A, B, C... provided, each with annotation.

-   Spec Type, Description, Alt Text included if visual used.

-   Turn & Talk included.

**TE**

-   Begins with **START** boilerplate + Three Reads.

-   TG Qs match each SE sub-item (≤5 each).

-   Includes **NEXT: Texas Process Standard** citation (verbatim).

-   Ends with **NOW, have students solve the problem**.

-   Supports: Turn & Talk TG, MLR routine, ML/UDL/Common Error aligned
    to item.

-   Task 1 only: DOK Level 1, 2, 3 Qs included with annotations.

**7. Practice On Your Own (POYO)**

**SE**

-   Interactions labeled (Interaction 1, 2...).

-   Each includes SE Manuscript + annotation.

-   Spec Type + Description included if visual.

-   Bold math vocabulary.

**TE**

-   Includes **Item Guide** (Item #, DOK, Task Alignment).

-   Includes at least one **Rubric** (2/1/0 points + sample answers).

-   Rubric matches constructed-response/model item.
