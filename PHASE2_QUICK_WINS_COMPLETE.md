## Lessons Learned

### 1. Consortium Identification Critical:

**Discovering WIDA consortium membership early in the process enabled:**
- Streamlined file creation (reference files instead of full guides)
- Massive future scalability (30+ additional WIDA states can reference IL guide)
- Reduced cognitive load (consistent framework across 40+ states)

**Recommendation:** Always identify multi-state consortiums first when expanding.

---

### 2. Reference File Pattern Works Well:

**Pattern:**
- Create 1 comprehensive guide for consortium (IL for WIDA, WA for ELPA21)
- Create short reference files for member states pointing to comprehensive guide
- Include state-specific nuances (assessment name, proficiency level labels)

**Benefits:**
- DRY principle maintained (single source of truth)
- Easy updates (update comprehensive guide, references stay valid)
- Clear navigation for users

---

### 3. State Adoption Processes Vary:

**Three Models Observed:**

**Local Control (7 states):**
- NY, IL, NJ, PA, OH, MI, OR
- No formal state adoption list
- Districts independently evaluate and select
- Lowest compliance barrier

**State Guidance (2 states):**
- WA, CO
- State reviews and rates materials
- Districts select from reviewed/recommended materials
- Moderate compliance

**Formal State Adoption (1 state):**
- NC
- State creates approved adoption list
- Districts select from approved list only
- Highest compliance barrier (similar to TX, CA, FL)

**Implication:** Materials designed for formal adoption states (TX, CA, FL) easily meet requirements for local control states.
